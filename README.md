# Project Name: Secure Note 

project overview: the secure note application mantain highly security, your data can't see server owner also, your data are encrypted by your private key we dose not sotore your encryption key,
we just store your encrypted/hash data.


### features
1. Dark/light mode
2. Local authentication (biometric, passwprd),
3. Task(pending, time-out, completed)
4. StopWatch
5. Note
6. Edit, update, delete
7. Local to Cloude/Vault
8. Vault/Cloude to Local
9. Save passkey in valut
